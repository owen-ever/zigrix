# Zigrix Legacy Python Prototype

이 디렉토리는 Zigrix의 **이전 Python prototype/reference 구현**을 보관한다.

원칙:
- 새로운 기능 개발은 여기서 진행하지 않는다.
- 동작 비교, 마이그레이션 참고, parity 체크 용도로만 유지한다.
- 현재 메인 구현체는 repository root의 **Node/TypeScript CLI** 다.

포함 범위:
- `src/` — 기존 Python CLI 구현
- `tests/` — Python 테스트
- `pyproject.toml` — 이전 Python 패키징 메타데이터
