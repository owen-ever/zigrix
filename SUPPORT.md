# Support

## Getting help
Start with these docs first:
- `README.md`
- `docs/quickstart.md`
- `docs/troubleshooting.md`
- `docs/cli-spec.md`
- `docs/release-readiness.md`

## Before asking for help
Please include:
- Zigrix version
- Node version
- OS
- install method (source checkout / release asset / npm package)
- output of `zigrix doctor`
- output of `zigrix state check --json` when runtime drift is suspected

## Where to ask
- usage questions: GitHub Discussions or issue tracker (when enabled)
- reproducible defects: use the bug report issue template
- security-sensitive reports: follow `SECURITY.md`, do not post publicly first

## Support boundary
Zigrix currently supports the local file-backed CLI flow on macOS/Linux first. OpenClaw integration is supported as an optional companion path, not a mandatory runtime dependency.
